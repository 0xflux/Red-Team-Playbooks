## Alternate Data Stream Files (ADS)

```c
C:\> echo <PAYLOAD> > %USERPROFILE%\AppData:<FILE>
```

### Listing ADS Files

```c
C:\> dir /r /a %USERPROFILE%\AppData\
```

### Deleting ADS Files

> https://live.sysinternals.com/

```c
C:\> .\streams64.exe -s C:\PATH\TO\FOLDER\
C:\> .\streams64.exe -d C:\PATH\TO\FOLDER\
```
